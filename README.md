# Supervised Learning Models using Scikit-Learn

## Overview

This project demonstrates how to build and compare multiple supervised machine learning models using Scikit-Learn.

## Algorithms

- Logistic Regression
- Decision Tree Classifier
- Support Vector Machine (SVM)

## Dataset

Iris Dataset (Scikit-Learn)

## Workflow

- Load dataset
- Train/Test Split
- Feature Scaling
- Train Models
- Evaluate Performance
- Compare Accuracy

## Libraries

- pandas
- numpy
- matplotlib
- scikit-learn

## Output

- Accuracy Score
- Classification Report
- Accuracy Comparison Chart
